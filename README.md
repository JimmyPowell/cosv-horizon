## 项目概述

COSV-Horizon 是一个集开源漏洞数据披露、精化于一体的在线平台，旨在规范漏洞信息管理，助力中国开源事业和漏洞安全防护事业。平台支持基于 COSV（CCF 开源漏洞信息描述规范）的标准化漏洞信息管理，使漏洞数据能够便捷地跨系统流转。

## 技术栈

### 前端
- Vue3
- TypeScript
- Tailwind CSS
- Element UI

### 后端
- Spring Boot
- MyBatis
- MySQL
- Docker
